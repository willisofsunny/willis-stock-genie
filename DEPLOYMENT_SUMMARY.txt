  $ git remote add origin https://github.com/YOUR_USERNAME/fingenius.git

步驟 3️⃣ : 在 Render 部署
  1. 訪問 https://dashboard.render.com

  5. 點擊部署！

🔑 API 密鑰設置
================================================================================

獲取密鑰：
  ✓ DeepSeek: https://platform.deepseek.com
  ✓ OpenAI:   https://platform.openai.com/api-keys

在 Render 中設置：
  1. 進入 Service → Environment
  2. 添加 DEEPSEEK_API_KEY
  3. 添加 LOG_LEVEL = INFO
  4. 保存並重啟

⚠️ 安全提示：
  - 千萬不要在代碼中暴露密鑰
  - 所有密鑰都用 Render Environment 管理
  - .env 文件已添加到 .gitignore
